Linguagens de Programação

Nome Características Pricipais aplicações Empresas que usam Opinião
Javascript Trabalha com o HTML e o CSS para o funcionamento de páginas web e permite a implementação de itens complexos nas mesmas Páginas web, aplicativos mobile e jogos Google, Facebook, Amazon, Netflix, Uber e Microsoft Ela parece ser bastante interessante dentre o Front-end, porém ainda prefiro HTML a Javascript
Java Orientação ao objeto, portátil, alta perfomance, segura e compilada Desenvolvimento de jogos, Big Data, IA, computação em nuvem e Internet das Coisas Spotify, Uber, Glovo, Lyft e Bitpanda É uma linguagem que, apesar de ter uma fama de ser difícil, decidi me arriscar e aprender um pouco, e até o momento tem sido bastante proveitoso
PHP Linguagem server-side, exige poucos recursos para funcionar e é capaz de gerar arquivos complexos Páginas Web, manipulação de arquivos, desenvolvimento de APIs e geração de arqivos XML e Json Facebook, Dafiti, Yahoo, Accenture, PicPay Não entendo muito sobre, por isso, prefiro não opinar, mas tenho interesse em aprender mais sobre PHP
Python Linguagem open source, multiplataforma, imperativa, de alto nível e interpretada Desenvolvimento web, automação, testagem de softwares, Machine Learning e análise de dados Instagram, Google, Spotify, Uber, Globo e Amazon Já tentei fazer alguns códigos porém não me aprofundei muito, mas sem dúvida é uma linguagem muito importante
C# Sintaxe simples, criada a partir da junção de linguagens como Java e C++ e orientação a objetos Plataformas Web, mobile, aplicações Desktop e desenvolvimento de jogos Microsoft, NASA, Intel, Oracle, Google e Paradox Por ser uma linguagem usada em muitos jogos, ela me despertou bastante interesse, pois tenho muita curiosidade sobre o desenvolvimento de jogos
Ruby Orientação a objetos, multiplataforma, open source, une leitura amigável e complexidade técnica Análise de dados, aplicações web, e aplicativos Airbnb, Twitch, Shopify, Esty Ela me despertou bastante o interesse por ser uma linguagem simples, que teve seu auge e queda, mas que agora tem voltado a ter mais relevância no mercado
